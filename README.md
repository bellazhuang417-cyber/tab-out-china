# Tab Out — 国内浏览器适配版

> 基于 [zarazhangrui/tab-out](https://github.com/zarazhangrui/tab-out) 及其 [Pinned fork](https://github.com/bellazhuang417-cyber/tab-out-pinned) 改造。

适配千问浏览器、夸克浏览器、360浏览器等国产 Chromium 浏览器。

---

## 为什么国内版不能直接接管新标签页？

原版 Tab Out 使用 Chrome Extension API 的 `chrome_url_overrides.newtab` 来替换浏览器新标签页。这个方案在 Chrome、Edge 等标准 Chromium 浏览器上完全正常，但在国内浏览器上**不生效**。

原因很简单：**国产浏览器锁死了新标签页的控制权。**

千问、夸克、360 等浏览器的新标签页是它们的**核心商业页面**——上面有搜索框、资讯信息流、快捷入口、广告位等。这是它们最重要的流量入口和变现渠道，所以它们在浏览器层面拦截了插件的 newtab 覆盖请求。

| 浏览器 | 是否支持插件覆盖新标签页 | 原因 |
|--------|------------------------|------|
| Chrome / Edge | ✅ 支持 | 新标签页被视为用户可定制空间 |
| 千问浏览器 | ❌ 不支持 | 新标签页是信息流+搜索的商业入口 |
| 夸克浏览器 | ❌ 不支持 | 同上 |
| 360 浏览器 | ❌ 不支持 | 同上 |
| UC 浏览器 | ❌ 不支持 | 同上 |

**解决方案：** 本插件改为 **Popup 弹出面板**模式——点击工具栏上的 Tab Out 图标即可弹出标签管理面板，不依赖浏览器是否允许覆盖新标签页。

## 与原版的主要差异

| 项目 | 原版 | 国内版 |
|------|------|--------|
| 启动方式 | 自动替换新标签页 | 点击扩展图标弹出 Popup 面板 |
| 字体 | Google Fonts (DM Sans + Newsreader) | 系统字体（苹方/微软雅黑/宋体） |
| Favicon | Google s2/favicons API | 彩色字母头像（兼容所有国产浏览器） |
| UI 文案 | 英文 | 中文 |
| 问候语 | Good morning/afternoon/evening | 早上好/中午好/下午好/晚上好/夜深了 |
| 日期格式 | English (Friday, April 4, 2026) | 中文（2026年4月4日星期五） |
| 浏览器内部页过滤 | chrome:// edge:// brave:// | + quark:// uc:// qqbrowser:// 等 11 种 |
| 国内域名映射 | 少量 | 60+ 国内常用网站（B站/知乎/微博/小红书/抖音/淘宝/飞书/钉钉等） |

## 安装

1. 下载或 clone 本仓库
2. 打开浏览器的扩展管理页面
   - 千问浏览器：菜单 → 扩展程序（或地址栏输入 `chrome://extensions/`）
   - 夸克浏览器：菜单 → 扩展程序
   - 360浏览器：地址栏输入 `se://extensions`
3. 开启**开发者模式**
4. 点击**加载已解压的扩展程序**，选择 `extension/` 文件夹
5. 点击工具栏上的 **Tab Out 图标** 即可弹出面板

> **注意：** 安装后新标签页不会有任何变化。请点击浏览器工具栏上的 Tab Out 图标来使用。

## 使用说明

### Popup 快捷面板（点击图标即弹出）
- 查看当前所有打开的标签页（按域名分组）
- 单独关闭某个标签页（悬停显示 × 按钮）
- 一键关闭全部标签页
- 固定快捷栏：自动从历史记录推荐常用网站

### 完整 Dashboard 面板
- 点击 Popup 面板底部的「打开完整面板」按钮
- 在新标签页中打开完整的 Tab Out 仪表盘
- 支持稍后阅读、归档搜索、重复标签检测等完整功能

---

## 功能

- **按域名分组显示**所有打开的标签页
- **首页分组**：B站/知乎/微博等首页单独归组，方便一键清理
- **固定快捷方式栏**：最多 8 个常用网站，一键直达
- **智能推荐**：根据最近 30 天浏览频率推荐固定
- **重复标签检测**：自动标记重复页面，一键去重
- **稍后阅读**：保存标签页到清单，带归档和搜索
- **100% 本地**：无服务器、无账号、无数据外传

## 自定义配置

编辑 `extension/config.local.js` 文件，取消注释并填入你的常用网站快捷方式。示例已写在文件中。
